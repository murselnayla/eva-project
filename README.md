# Eva Project
--

## Tech Stack

- Vue.js 3 (Composition API)
- Vite
- Pinia
- TypeScript
- Tailwind
- Highcharts


## Project Setup

```sh
git clone https://github.com/murselnayla/eva-project.git
```

```sh
cd eva-project
```

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev

http://localhost:5173
```

### Type-Check, Compile and Minify for Production

```sh
yarn build 
```

### Lint with [ESLint](https://eslint.org/)

```sh
yarn lint
```
